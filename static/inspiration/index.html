Mission: Decentralize the Internet

As fundamental freedoms are continually encroached upon, having a decentralized and democratic internet is more important than ever before. Unfortunately, the current state of progress for internet decentralization is a big fragmented mess of different projects trying to do different things, but not effectively enough. Not enough ideas are shared, not enough people are talking. The biggest usage of peer-to-peer tech is Bittorrent, which has been around for 2 decades. Other parts of internet decentralization have been mostly re-centralized by necessity for ease of use (such as LBRY or Ethereum). This fragmented ecosystem is in desperate need of one project that can replace all the necessary parts of the internet so that reliance on centralized services is not needed anymore.

Projects that have inspired Dither

Multiformats - Multiformats aims to fix the problem of future-proofing systems into future ages of computing. Dither will use a few of the multiformats, such as multihash, to future-proof and to be compatible with IPFS.

Stellar Conensus Protocol - Byzantine Federated Voting is a really fast and flexible system for consensus, it just has to be implemented well.

Monero - Monero is the only cryptocurrency that actually has most of what a crypto currency should have. Transactions are completely anonymous, transaction fees are very low and it's not managed by a single company. Dither will improve on this with its own suite of cryptocurrencies that are all completely anonymous and take advantage of the Stellar Consensus Protocol for blazing fast transaction speeds.

IPFS - IPFS is a core project in the current stage of Web3, and a really good start to a decentralized internet. But it is simply too slow and hard to interface with. (The Precarious State of IPFS) - Dither's Directional Trail Search will greatly improve the speed at which data can flow across the network. It also improves on IPFS by allowing for permissioned data access and more flexible ways of accessing data.

DNS - The Domain Name System is structurally decentralized, but it is centrally managed by ICANN, an organization with the authority to control domain name registration. Dither will have its own naming system that will replace DNS and allow for much better systems of naming things. This naming system will be built on the Reverse Hash Lookup system.

TOR & I2P - These "Hidden Networks" are very important for obfuscating connections between people on the internet. However, they are very slow due to their nature of randomly routing connections through different nodes. Dither will improve on this with Distance-Based Routing , a routing system that takes latency into account when creating tunnels, allowing for the user to choose between better privacy or better speed. Dither will also implement a technology called HORNET to allow for addition efficiency when establishing obfuscated connections

Other notable projects are: Nomia, IPLD, IOTA, zk-STARKS, and Solidity.
