Mission: Decentralize the Internet

The current state of decentralizing the internet is a big fragmented mess of different protocols and applications trying to do different things, but not effectively enough. Not enough ideas are shared, not enough people are talking. The biggest usage of peer-to-peer tech is bittorrent, which has been around for 2 decades. As freedoms are continually encroached upon, having a decentralized and democratic internet is more important than ever before.

Projects That have inspired Dither

Multiformats - Multiformats aims to fix the problem of future-proofing systems into future ages of computing. Dither will use a few of the multiformats, such as multihash, to future-proof and to be compatible with IPFS

Stellar Conensus Protocol - Byzantine Federated Voting is a really fast system for consensus, it just has to be implemented well.

Monero - Monero is the only cryptocurrency that actually has most of what a crypto currency should have. Transactions are completely anonymous, transaction fees are very low and it's not managed by a single company. Dither will improve on this with its own suite of cryptocurrency

IPFS - IPFS is a core project in the current stage of Web3, and a really good start to a decentralized internet. But it is simply too slow and hard to interface with. (The Precarious State of IPFS) - Dither's Directed Trail Search will greatly improve the speed at which data can flow across the network. It also improves on IPFS by allowing for permissioned data access.

DNS - The Domain Name System is a very old piece of the internet, but it is unfortunately centralized. There have been various attempts at block-chainifying. A replacement system for naming things will be built on top of the Reverse Hash Lookup system.
